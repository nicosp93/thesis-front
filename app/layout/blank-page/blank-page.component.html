About

The goal of this project is to develop a web application implementing a dashboard able to show data received from the devices connected to it.

These devices are going to be placed all over the Universitat Aut√≥noma de Barcelona (UAB) to send data like humidity, temperature, brightness and more using Low-Power Wide Area Network Protocol (LPWAN). These data is going to be stored in a server for its further treatment.

Every time a message is sent to this server, my application will be listening in order to store this data in a database and show this data in the web application I implemented.

Objectives

As said, the main objective of this project is to develop a web application implementing a dashboard able to show data received from the devices connected to it. To achieve this objective I went through a set of sub objectives:

  • Develop a software solution able to manage and show the data captured by the devices in a given time frame.
  • Study the currently available technologies and find out which are more suitable to reach the main objective.
  • Apply LoRa protocol in a real environment to show how this protocol interacts with other technologies in a real environment.
